🎵 Chinook SQL Database Analysis
📌 Project Overview

This project was completed as part of Week 3 of my Data Analytics Internship at AnalystLabAfrica.

The objective was to analyze the Chinook Music Store Database using SQL and generate business insights from a relational database. The project demonstrates practical SQL skills, including querying multiple related tables, data aggregation, joins, subqueries, and window functions.

🛠 Tools Used
Microsoft SQL Server 2022
SQL Server Management Studio (SSMS)
Chinook Database
SQL
📂 Database Structure

The Chinook database contains 11 relational tables, including:

Artist
Album
Track
Customer
Employee
Invoice
InvoiceLine
Genre
Playlist
PlaylistTrack
MediaType

These tables are connected using primary and foreign keys to simulate a real-world digital music store.

📈 Key Findings
Helena Holý was the highest-spending customer.
The USA generated the highest total revenue.
Rock was the best-selling music genre.
Iron Maiden was the top-selling artist.
"Minha Historia" was the highest-selling album.
MPEG Audio Files were the most purchased media type.
Jane Peacock supported the highest number of customers.
